Design Patterns - Drawing Program


A WordArt like drawing program made with C#. This program is made by using multiple design patterns. The design patterns that are used for this program are:

  • Command Pattern; for unlimited undo and redo functionality.
  • Composite Pattern; to put multiple figures into the same group.
  • Visitor Pattern; for replacing, resizing and saving data.
  • Strategy Pattern and Singleton Pattern; for drawing the figures.
  • Decorator Pattern; for adding captions.
  • Language: C#
  • Source: Google Drive
  • Build: Google Drive

Promotional Easter game


Previous Next

A promotional game with an Easter theme which I created during my internship at 8D-Games. The game was created with C# in the Unity3D engine. I created this game together with another intern who created all the sprites for the game. And I was responsible for all the technical aspects of it.

Serious Game - Ice Skating


Previous Next

A Serious Game in which elderly people can practice their balance and prevent injuries, by playing a game about ice skating in combination with Microsoft's Kinect.

I got to create an update tool for the game during my internship at 8D-Games. It will automatically check for updates when the user starts the game. And give the user the option to download the latest version.

​

To bring more variety to the game I also created the possibility to place markers on the tiles that the game consists of. For each marker the developer can select multiple objects that can spawn at those locations and also set their direction. This feature gives the user always another experience when they play the game.

  • Language: C#
  • Engine: Unity

Port of Rotterdam Simulation


Previous Next

For a project during my second year at the NHL Hogeschool we created a simulation of the port of Rotterdam, together with five other students. The project consisted of a 3D simulation and a server that would send XML data to the simulator. My main tasks consisted of:
