Design Patterns - Drawing Program


A WordArt like drawing program made with C#. This program is made by using multiple design patterns. The design patterns that are used for this program are:

  • Command Pattern; for unlimited undo and redo functionality.
  • Composite Pattern; to put multiple figures into the same group.
  • Visitor Pattern; for replacing, resizing and saving data.
  • Strategy Pattern and Singleton Pattern; for drawing the figures.
  • Decorator Pattern; for adding captions.
  • Language: C#
  • Source: Google Drive
  • Build: Google Drive

Serious Game - Ice Skating

A Serious Game in which elderly people can practice their balance and prevent injuries, by playing a game about ice skating in combination with Microsoft's Kinect.

I got to create an update tool for the game during my internship at 8D-Games. It will automatically check for updates when the user starts the game. And give the user the option to download the latest version.

​

To bring more variety to the game I also created the possibility to place markers on the tiles that the game consists of.
